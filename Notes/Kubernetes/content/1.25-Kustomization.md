## 1.25 Kustomization

Nos permite generar manifiestos a partir de un archivo yaml.

```yaml
apiVersion: Kustomize.config.k8s.io/v1beta1
kind: Kustomization

commonLabels:
  app: <common-label>

resources:
- <manifest_file>.yaml

secretGenerator
- name: <name>
  literals:
  - <key>=<value>
  - <key>=<value>
```

Ya ahora podemos obtener el resultado con el comando kustomize build.

```bash
kustomize build .
```
