## 1.4 Modelo aditivo RGB

Es un modelo donde cada color se representa con un vector de 3
dimensiones (R, G, B)

El color rojo se forma con (255, 0, 0) El color verde con (0, 255, 0) El
color azul con (0, 0, 255)
