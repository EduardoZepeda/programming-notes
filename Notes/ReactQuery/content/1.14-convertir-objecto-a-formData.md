## convertir un objecto en formData

Para convertir un objecto en formData, bastante Ãºtil para subir archivos a django rest framework directamente, en lugar de usar base64 podemos echar mano de Javascript en el frontend
 
 ``` javascript
const convertToFormData = (data: <form>): FormData => {
   const formData = new FormData()
   formData.append('<campo_1>', data.campo1)
   formData.append('<campo_2>', data.campo2)
   if (data.<imagen> && data.<imagen>.length > 0) {
       formData.append('<imagen>', data.<imagen>[0])
   }
   return formData
}
 ```
